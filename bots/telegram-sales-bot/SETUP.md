# 🤖 نظام الرد على تيليجرام (بموافقتك) — دليل التركيب

نظام مجاني ١٠٠٪: **بنك إجابات بكلمات مفتاحية** (بدون AI). العميل يبعت → النظام يلقط الكلمة → يبعتلك الرد للمراجعة بزرّين [✅ موافقة] [✏️ تعديل] → ما يتبعتش للعميل غير بموافقتك.

> ⭐ **مهم:** استخدم نسخة **Cloudflare Workers** (`worker.js`) — دي الموثوقة. نسخة Google Apps Script القديمة كانت فيها مشكلة إن تيليجرام بيقف بعد أول رسالة (بسبب رد 302)، وتم حذفها من الريبو.

# 🟢 الطريقة الموصى بيها: Cloudflare Workers (worker.js)

## الخطوة ١: اعمل البوت
- كلّم **@BotFather** → `/newbot` → خد الـ **TOKEN**.
- خد رقمك (Chat ID) من **@userinfobot**. واضغط **Start** على البوت بتاعك.

## الخطوة ٢: اعمل Worker على Cloudflare (مجاني)
1. سجّل في **dash.cloudflare.com** (مجاني، بدون كارت).
2. **Workers & Pages → Create → Workers → Create Worker** → سمّيه (مثلاً empire-bot) → **Deploy**.
3. اضغط **Edit code** → امسح الكود الافتراضي → الصق محتوى **worker.js**.
4. فوق الكود، حط التوكن ورقمك في أول السطور:
   `const TELEGRAM_TOKEN = "..."` و `const ADMIN_CHAT_ID = "..."`.
5. **Deploy**. وانسخ رابط الـ Worker (شكله: `https://empire-bot.xxxx.workers.dev`).

## الخطوة ٣: وصّل البوت
افتح ده في المتصفح (بدّل التوكن ورابط الـ worker):
```
https://api.telegram.org/bot<التوكن>/setWebhook?url=<رابط_الworker>&drop_pending_updates=true
```
لازم يرجّع `{"ok":true,...}` ✅

## جرّب
- كلّم البوت **من حساب تاني** واسأل «الباقات إيه؟».
- المفروض **يوصلك إنت** الرد المقترح + زرّين → اضغط ✅ يروح للعميل.
- ابعت رسالة تانية وتالتة — هيفضل شغّال من غير ما يقف 🔥 (لأن Cloudflare بيرجّع 200).

## الخطوة ٤: فعّل التعلّم التلقائي 🧠 (KV) — مهم
عشان البوت يحفظ ردودك على الأسئلة الجديدة ويرد بيها تلقائيًا المرة الجاية:
1. من صفحة الـ Worker → **Settings → Bindings → Add binding → KV namespace**.
2. **Create a namespace** باسم مثلاً `empire-bank`.
3. خلي **Variable name** بالظبط: `KV` → اختَر الـ namespace → **Save/Deploy**.
> من غير الخطوة دي البوت يشتغل عادي، بس مش هيتعلم تلقائيًا (هيفضل يحوّلك كل سؤال جديد).

### إزاي التعلّم بيشتغل؟
- سؤال جديد مالوش رد → يوصلك بزر **🧠 رد + تعليم** → تكتب الرد → يتبعت للعميل **ويتحفظ**.
- أول ما حد يسأل نفس السؤال تاني → البوت يقترحلك نفس الإجابة جاهزة للموافقة ✅.

➕ لإضافة سؤال جاهز يدويًا: في `worker.js` ضيف عنصر في `ANSWERS`.

## الخطوة ٥: فعّل التذكيرات اليومية ⏰ (Cron)
عشان البوت يبعت تذكيرات تلقائية للي ماشتركش + يطلب فيدباك من المشتركين:
1. صفحة الـ Worker → **Settings → Triggers → Cron Triggers → Add**.
2. اكتب: `0 16 * * *` (يوميًا ٤ العصر UTC = ٦م القاهرة / ٨م دبي) → **Save**.
> أقصى ٤ تذكيرات للعميل، وبتوقف أول ما يشترك. وبيطلب فيدباك بعد يومين من الاشتراك.

## أوامر الأدمن:
- `/version` (لازم v8) · `/kv` (الذاكرة) · `/list` (المتعلّم) · `/stats` (عملاء/مشتركين/إيصالات)

## ميزات v8:
- 🧾 العميل يبعت صورة إيصال → بتتحوّل لك مع زر **✅ تأكيد الاشتراك** (بيرحّب بيه ويوقف التذكيرات).
- 🧠 ذاكرة لكل عميل (فاكر واقف فين) + تذكيرات حسب مرحلته.
- 💬 ردود تسويقية محفّزة + upsell/cross-sell.
