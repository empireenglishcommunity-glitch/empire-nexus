## System Overview

Empire English هو نظام تعلّم إنجليزي يومي للعرب. كل يوم الساعة 6 الصبح
(توقيت دبي) بتنزل 7 مهام. الطالب بيعمل المهام ويكتب !done عشان يسجل.

## Daily Tasks (7 tasks per day)

1. **Accent Drill** (!done accent) — سجل صوتك في #l0-showcase ثم اكتب الأمر
2. **Vocabulary** (!done vocab) — البوت هيسألك سؤال، جاوب صح
3. **Shadowing** (!done shadow) — سجل 30 ثانية محاكاة في #l0-showcase
4. **Speaking** (!done speaking) — سجل صوتك (المدة حسب المستوى)
5. **Listening** (!done listening) — البوت هيسألك سؤال فهم
6. **Writing** (!done writing) — اكتب 10 حروف+ في #l0-text-practice
7. **Community** (!done community) — اكتب في #general-chat أو ادخل فويس 10 دقايق

## How to Submit Tasks

1. اعمل المطلوب (سجل صوت، اكتب، إلخ)
2. اكتب `!done accent` (أو اسم المهمة)
3. أو اكتب `!1` لحد `!7` (رقم المهمة)
4. أو اضغط الإيموجي 1️⃣-7️⃣ على رسالة المهام

## Common Problems + Solutions

### "مش عارف أسجل المهمة"
- لازم ترفع تسجيل صوتي في #l0-showcase الأول
- بعدين ارجع اكتب `!done accent` في #bot-commands
- التسجيل لازم يكون آخر ساعتين

### "البوت مش بيرد"
- اكتب الأوامر في #bot-commands (مش في أي قناة تانية)
- الأوامر لازم تبدأ بـ ! (مثلاً !done مش done)
- ممكن يكون عندك cooldown (5 دقايق بين كل !done)

### "مش فاهم المهام"
- كل يوم 7 مهام بتنزل الساعة 6 صباحاً
- شوف #l0-daily-tasks
- كل مهمة فيها تفاصيل كاملة
- لو مش فاهم مهمة معينة، اسأل وأنا هشرح

### "عايز أعرف مستواي فين"
- اكتب `!progress` أو `!تقدم`
- هتشوف: المستوى، النقاط، الـ streak، المهام

### "مش عارف أسجل صوت على الموبايل"
- اضغط على + في Discord (جنب مكان الكتابة)
- اختار "Voice Message" أو "Upload File"
- سجل أو ارفع ملف صوتي
- لازم يكون في قناة #l0-showcase

### "الـ streak بتاعي اتكسر"
- الـ streak بيتكسر لو عدى يوم من غير ما تعمل حتى مهمة واحدة
- لو عايز تحافظ عليه، اعمل مهمة واحدة على الأقل كل يوم
- أسهل مهمة: اكتب جملة في #general-chat واكتب `!7`

## Commands (all work in Arabic too)

| Command | Arabic | What it does |
|---------|--------|-------------|
| !done <task> | !تم <مهمة> | Mark task complete |
| !1 to !7 | !١ to !٧ | Quick task submit by number |
| !progress | !تقدم | View progress dashboard |
| !streak | !سلسلة | View streak info |
| !level | !مستوى | Level + advancement info |
| !week | !أسبوع | This week's curriculum |
| !portfolio | !صوتي | Voice recording history |
| !words | !كلماتي | SRS vocabulary review |
| !abilities | !قدراتي | Milestone tracker |
| !help | !مساعدة | All commands |
| !link | !ربط | Connect practice platform |
| !difficulty | !صعوبة | View adaptive difficulty |
| !notifications | !إشعارات | Notification preferences |

## Practice Platform
