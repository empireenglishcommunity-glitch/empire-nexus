# 📚 دليل المحتوى — Content Index

> فهرس لكل ملفات المحتوى الخاصة بتحدّي 30 يوم، ومتى تستخدم كل ملف.
> An index of all content files for the 30-day challenge, and when to use each.

---

## 🗂️ ملفات المحتوى (Content Files)

| الملف | المحتوى | متى تستخدمه |
|-------|---------|--------------|
| `challenges.json` | الـ 30 تحدّيًا (بيانات يقرأها البوت) | يقرأه البوت تلقائيًا كل يوم |
| `launch-week-promo.md` | 7 سكربتات فيديو تشويقية | الأسبوع الذي **قبل** اليوم 1 |
| `launch-day-live-script.md` | سكربت لايف الانطلاق (عربي/إنجليزي) | **اليوم 0** — جلسة تحديد الأهداف |
| `tiktok-captions.md` | كابشنات TikTok بالعربية (30 يوم) | يوميًا أثناء التحدّي |
| `tiktok-captions-en.md` | كابشنات TikTok بالإنجليزية (30 يوم) | يوميًا (لجمهور أوسع) |
| `poster-text.md` | نصوص الجرافيك/البوسترات (30 يوم) | لتصميم بوستر كل يوم |

---

## 🔄 تسلسل الإطلاق (Launch Flow)

```
الأسبوع -1   →  launch-week-promo.md      (7 فيديوهات تشويق)
اليوم 0      →  launch-day-live-script.md (لايف تحديد الأهداف)
الأيام 1–30  →  tiktok-captions(.md/-en)  +  poster-text.md
              +  البوت ينشر التحدّي تلقائيًا من challenges.json
المراجعات    →  كل 7 أيام (لايف) — البوت: !recap
اليوم 30     →  لايف الاحتفال + شهادات (البوت: !cert)
```

---

## 🧩 كيف تترابط القطع (How It All Connects)

1. **قبل الإطلاق:** انشر فيديوهات `launch-week-promo` لبناء الحماس ووجّه الناس للديسكورد.
2. **يوم الانطلاق:** أدِر اللايف باستخدام `launch-day-live-script`، وسجّل المشاركين عبر `!join`.
3. **كل يوم:** البوت ينشر التحدّي من `challenges.json`؛ انشر الكابشن المطابق (عربي/إنجليزي) وبوستر اليوم.
4. **يتابع البوت** التقدّم والرتب ولوحة المتصدّرين تلقائيًا.
5. **النهاية:** لايف احتفال + شهادات PDF عبر `!cert`.

> كل المحتوى أصلي ومتناسق بصريًا ولغويًا عبر البوت والبوسترات والشهادات.
