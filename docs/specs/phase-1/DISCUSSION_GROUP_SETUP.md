# Empire English — Discussion Group Setup Guide

## Quick Setup (5 minutes)

### Step 1: Create the Group
1. Open Telegram → New Group
2. Name: **Empire English Community | مجتمع Empire**
3. Add @EmpireEnglishBot as admin
4. Set photo: your Empire brand image (black + gold)

### Step 2: Link to Channel
1. Open your Telegram Channel → Settings → Discussion
2. Select the newly created group
3. This auto-forwards channel posts to the group with a "Comment" button

### Step 3: Set Permissions
- Send Messages: Everyone
- Send Media: Everyone
- Add Members: Admins only
- Pin Messages: Admins only
- Invite via Link: Everyone

### Step 4: Pin the Rules Message
Copy and send the rules message below, then pin it.
