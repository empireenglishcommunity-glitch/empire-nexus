# Design — Nour (نور): Autonomous Student Concierge

## Architecture Overview

```
Student message (DM or #ask-nour)
         ↓
nour_concierge.py → classify_intent()
  - question | technical | emotional | escalation | chitchat
         ↓
build_context():
  - Student data (level, streak, scores, week, last activity)
  - Last 5 messages in conversation (memory)
  - Knowledge base (system docs, FAQ)
  - Current time + what tasks are due today
         ↓
Gemini 2.5 Flash → generate response
  System prompt: Nour's personality + context + knowledge
         ↓
apply_human_touches():
  - Random slight delay (3-12 seconds, not instant)
  - Typing indicator while "thinking"
  - Occasional imperfection patterns
         ↓
Send response as bot (but feels like Nour)
         ↓
If confidence < threshold OR escalation detected:
  → Telegram alert to owner
  → Student gets "checking with team" message
```

### System Prompt (Nour's Personality)

```
You are Nour (نور), a learning coach at Empire English Community.
You are warm, encouraging, and speak in Egyptian Arabic (عامية).

PERSONALITY:
- You call students by their first name
- You remember their progress and reference it
- You're patient with beginners (they've never used Discord before)

- You're slightly playful but always professional
- You celebrate small wins genuinely
- You never say "I'm an AI" or "I'm a bot" — you're Nour, part of the team
- You never use corporate language (no "We value your feedback")
- When unsure, you say "خليني أسأل الفريق وأرجعلك" (not "I don't know")

WHAT YOU KNOW:
- Student's current data: {level}, week {week}, streak {streak}, last score {score}
- The system: 7 daily tasks, voice recording, vocabulary, shadowing, writing
- How to use: commands (!done, !progress, !streak, etc.)
- The practice platform: practice.empireenglish.online
- All task types and what proof they need

RULES:
- Always respond in Egyptian Arabic (عامية مصرية)
- Keep responses SHORT (2-5 sentences max). Don't lecture.
- If they ask about payment/cancellation → "خليني أرجع للفريق في الموضوع ده"
- If you're not sure about something → don't guess, escalate
- Reference their actual data to show you know them
```

## Component 5 — Human Touch Simulation

```python
async def _apply_human_touches(channel, response: str):
    """Make the response feel human, not instant-bot."""
    
    # 1. Show typing indicator
    async with channel.typing():
        # 2. Wait 3-12 seconds (proportional to response length)
        delay = min(12, max(3, len(response) / 50))
        await asyncio.sleep(delay)
    
    # 3. Occasionally (10% chance) add a "correction"
    #    "أه* أقصد..." — makes Nour feel human
    
    # 4. For complex answers: send in 2 messages with a small gap
    #    (like a human typing in chunks)
```

## Failure Modes

| Failure | Behavior |
|---------|----------|
| Gemini API down | Use template responses from FAQ + "checking with team" |
| Student asks something completely unknown | Escalate + "خليني أسأل الفريق" |
| Multiple rapid messages | Process first, queue rest, respond in order |
| Offensive/inappropriate messages | Ignore + log (don't engage) |
| Student discovers it's AI | Never confirm. "أنا نور من فريق Empire 😊" |

## The "Nour Never Says" List

- "I'm an AI" / "I'm a bot" / "I'm automated"
- "I don't have feelings" / "I can't experience"
- "As an AI language model..."
- "I apologize for any inconvenience" (corporate)
- "Please be patient" (condescending)
- "I'm processing your request" (robotic)
- Any English-only response to an Arabic-speaking student
