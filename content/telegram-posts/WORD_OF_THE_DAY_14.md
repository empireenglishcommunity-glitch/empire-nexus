# Word of the Day — 14 Days (Daily C9 Ritual)

**Format:** Short Arabic-led post with pronunciation tip + reply CTA  
**Posting time:** Every day alongside the main post (or standalone on off-days)  
**Purpose:** Build daily engagement habit; invite replies in the discussion group

## Day 1: improve

```
📖 كلمة اليوم: improve

🔊 النطق: im-PROOV (التشديد على المقطع الثاني)
📝 المعنى: يتحسّن / يطوّر

✍️ مثال:
"I want to improve my pronunciation."
أبي أحسّن نطقي.

💬 استخدمها في جملة بالتعليقات 👇
```

📝 المعنى: يتمرّن / تمرين

## Day 12: pronounce

```
📖 كلمة اليوم: pronounce

🔊 النطق: prə-NOUNS (مو "بروناونس" — الشوا أول!)
📝 المعنى: ينطق

✍️ مثال:
"How do you pronounce this word?"
كيف تنطق هالكلمة؟

🔑 العائلة:
• pronounce (فعل) = ينطق
• pronunciation (اسم) = نطق ← prə-nun-see-AY-shən
⚠️ انتبه: ما فيها "pronounciation" — خطأ شائع!

💬 إيش أصعب كلمة إنجليزية تنطقها؟ 👇
```

## Day 13: fluent

```
📖 كلمة اليوم: fluent

🔊 النطق: FLOO-ənt (مقطعين)
📝 المعنى: طليق / يتكلم بطلاقة

✍️ مثال:
"I want to become fluent in English."
أبي أصير طليق بالإنجليزي.

🔑 العائلة:
• fluent (صفة) = طليق
• fluently (ظرف) = بطلاقة
• fluency (اسم) = طلاقة

⚡ نصيحة: الطلاقة مو = "ما تغلط أبدًا"
الطلاقة = تتكلم بسهولة وسلاسة، حتى لو بأخطاء بسيطة.

💬 كيف تعرّف "fluent" بالنسبة لك شخصيًا؟ 👇
```

## Day 14: consistent

```
📖 كلمة اليوم: consistent

🔊 النطق: kən-SIS-tənt (٣ مقاطع — شوا أول وآخر!)
📝 المعنى: مستمر / ثابت / منتظم

✍️ مثال:
"Being consistent is more important than being perfect."
الاستمرارية أهم من الكمال.

🔑 العائلة:
• consistent (صفة)
• consistently (ظرف) = بشكل مستمر
• consistency (اسم) = الثبات / الاستمرارية

⚡ هذي فلسفة Empire كلها:
١٥ دقيقة يوميًا > ٣ ساعات مرة بالأسبوع.
Consistency beats intensity. Always.

💬 إيش شي أنت consistent فيه بحياتك؟ (أي شي — مو بس إنجليزي) 👇
```
